# Transly 积分模型与定价策略 (2025年优化版)

## 1. 概述

本文件详细阐述了 Transly 平台优化后的虚拟积分系统和相关的定价策略。新策略以 $5 USD 作为入门价位，提供更合理的价格梯度和用户体验。

## 2. 积分模型 (Credit Model)

### 2.1 什么是积分？

- **定义**: "积分" (Credits) 是 Transly 平台内部使用的唯一虚拟货币。用户通过消耗积分来使用平台的各项付费服务。
- **价值**: 积分的设计与翻译的字符数量挂钩，为服务的使用提供了一个统一的度量单位。

### 2.2 积分价值与计费规则

- **免费翻译**: 500字符以下完全免费，无需消耗积分
- **付费翻译**: 超过500字符的部分按 `0.1积分/字符` 计费
- **最小扣费**: 50积分（即500字符以上的翻译最少扣费50积分）
- **计算公式**: `消耗积分 = MAX(50, (总字符数 - 500) × 0.1)`

### 2.3 计费示例
```
翻译字符数     免费部分    付费部分    消耗积分    等价费用
≤500字符      全部免费    0字符       0积分       $0
800字符       500字符     300字符     50积分      ~$0.10
2000字符      500字符     1500字符    150积分     ~$0.30
10000字符     500字符     9500字符    950积分     ~$1.90
```

## 3. 积分的获取

### 3.1 新用户激励（优化版）
- **注册奖励**: 500积分（约$2价值，可翻译约5万字符）
- **首次充值奖励**: 
  - Starter套餐($5): 额外赠送500积分(20%奖励)
  - Basic套餐($10): 额外赠送1200积分(20%奖励)  
  - Professional套餐($25): 额外赠送5000积分(25%奖励)
  - Business套餐及以上: 额外赠送25%积分

### 3.2 推荐计划
- **推荐人奖励**: 成功推荐获得300积分
- **被推荐人奖励**: 注册后额外获得200积分
- **双向激励**: 提升用户获取效率

### 3.3 忠诚度奖励
- **铜牌用户**($10-49累计消费): 充值时额外获得5%积分
- **银牌用户**($50-199累计消费): 充值时额外获得10%积分
- **金牌用户**($200-499累计消费): 充值时额外获得15%积分
- **钻石用户**($500+累计消费): 充值时额外获得20%积分

## 5. 月度订阅选项（新增）

为满足稳定需求用户，我们提供月度订阅服务：

| 订阅套餐 | 月费(USD) | 月度积分 | 年付优惠 | 相比一次性购买优势 |
|---------|----------|----------|----------|------------------|
| **基础月付** | $12 | 7,200积分 | 2个月免费 | +20%积分，自动续费 |
| **专业月付** | $30 | 25,000积分 | 2个月免费 | +25%积分，优先支持 |
| **企业月付** | $80 | 70,000积分 | 2个月免费 | +40%积分，专属服务 |

### 5.1 订阅优势
- **积分奖励**: 订阅用户比一次性购买多获得20-40%积分
- **自动续费**: 无需手动充值，确保服务连续性
- **优先支持**: 订阅用户享受优先客服支持
- **灵活取消**: 可随时取消，剩余积分保留

## 6. 特殊政策

### 6.1 积分有效期
- **购买积分**: 永久有效，无过期时间
- **赠送积分**: 有效期90天（注册奖励、推荐奖励等）
- **订阅积分**: 订阅期内有效，取消后保留剩余积分
- **消耗顺序**: 优先消耗即将过期的积分

### 6.2 退款政策
- **时间限制**: 购买后14天内可申请退款
- **使用限制**: 仅未使用的积分可退款
- **处理流程**: 通过客服渠道提交申请，3-5个工作日处理
- **退款方式**: 原路退回到支付账户

### 6.3 价格调整政策
- **已购积分**: 价格调整不影响已购买的积分
- **预告期**: 价格上调前30天通知用户
- **保护期**: 老用户享受30天价格保护期

## 7. 转化策略

### 7.1 智能推荐系统
- **使用量追踪**: 自动分析用户翻译习惯
- **套餐建议**: 基于历史数据推荐最适合套餐
- **成本优化**: 显示不同套餐的成本效益对比
- **时机把握**: 在积分余额不足时推荐充值

### 7.2 渐进式引导
1. **新用户期**(1-3天): 完全免费体验，建立使用习惯
2. **认知期**(4-7天): 显示积分概念，解释计费规则
3. **转化期**(8天+): 积分不足时推荐合适套餐
4. **留存期**: 通过忠诚度计划和订阅服务提升留存
